# Design Tokens

Design tokens are the visual design atoms of the design system â€” specifically, they are named entities that store visual design attributes. Venomous UI provides design tokens as CSS custom properties (variables).

## Usage

### CSS Variables (Recommended)

Import the CSS variables file and use them directly in your styles:

<Source
  language="css"
  code={`/* Import CSS variables */
@import "venomous-ui/core/styles/variables/index.css";

/_ Use in your CSS _/
.my-component {
color: var(--VENOMOUSUI-text-primary);
padding: var(--VENOMOUSUI-spacing-md);
border-radius: var(--VENOMOUSUI-radius-md);
}`}
/>

### JavaScript/TypeScript

For inline styles or CSS-in-JS, use the `designTokens` object:

<Source
  language="tsx"
  code={`// IMPORTANT: Import CSS variables first
import "venomous-ui/core/styles/variables/index.css";
import { designTokens } from "venomous-ui/core/design-tokens";

// React

<div
  style={{
    color: designTokens.colors.primary,
    padding: designTokens.spacing.md,
    borderRadius: designTokens.radius.md,
  }}
>
  Hello World
</div>

// Vue

<div :style="{
  color: designTokens.colors.primary,
  padding: designTokens.spacing.md,
}">
  Hello World
</div>`}
/>

### Runtime Value Access

Use `getCssVar` with `CSS_VAR_NAMES` to get the computed numeric value of a CSS variable at runtime. This is useful when you need the actual pixel value for calculations (e.g., canvas drawing, animations).

<Source
  language="tsx"
  code={`import "venomous-ui/core/styles/variables/index.css";
import { CSS_VAR_NAMES } from "venomous-ui/core/constants";
import { getCssVar } from "venomous-ui/core/helpers";

// Get spacing value as number (e.g., 12)
const spacingMd = getCssVar(CSS_VAR_NAMES.spacingMd);

// With fallback
const spacingSm = getCssVar(CSS_VAR_NAMES.spacingSm, 8);

// Get duration value for animations
const duration = getCssVar(CSS_VAR_NAMES.durationNormal);

// Use in calculations
const totalPadding = spacingMd \* 2;`}
/>

**Note:** `getCssVar` returns a parsed integer value, not a string. It's designed for numeric CSS variables like spacing, z-index, etc.
