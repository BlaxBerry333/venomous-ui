# CSS Classes

Venomous UI uses a consistent BEM-like naming convention for all CSS classes:

<Source
  language="css"
  dark
  code={`/* Block — component root */
.venomous-button { }

/_ Modifier — variant / size / state _/
.venomous-button--solid { }
.venomous-button--sm { }
.venomous-button--disabled { }

/_ Element — sub-part of a compound component _/
.venomous-card**header { }
.venomous-card**body { }`}
/>

All class names are available via the `CSS_CLASSES` constant for type-safe usage:

<Source
  language="ts"
  dark
  code={`import { CSS_CLASSES } from "venomous-ui/core/constants";

// Example
const cls = CSS_CLASSES.button.root; // "venomous-button"
const size = CSS_CLASSES.button.sizes.sm; // "venomous-button--sm"`}
/>
