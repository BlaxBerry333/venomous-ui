# CSS Classes

Venomous UI provides `CSS_CLASSES` constant object for type-safe CSS class name usage.

<StoryDocsTable
  headers={["Constant", "Purpose", "Example Value"]}
  rows={[["`CSS_CLASSES`", "HTML class names for components", `\`"${CSS_CLASSES.button.root}"\``]]}
/>

## Naming Convention

All CSS classes follow a predictable BEM-like pattern:

<Source
  language="css"
  dark
  code={`/* Block */
.venomous-button { }

/_ Modifier - variant _/
.venomous-button--solid { }
.venomous-button--outline { }

/_ Modifier - size _/
.venomous-button--sm { }
.venomous-button--lg { }

/_ Modifier - state _/
.venomous-button--disabled { }

/_ Element (for compound components) _/
.venomous-card**header { }
.venomous-card**body { }`}
/>

This makes it easy to:

- Find styles in DevTools
- Write targeted overrides
- Understand component structure at a glance

## Usage

<Source
  language="ts"
  dark
  code={`import { CSS_CLASSES } from "venomous-ui/core";

// Access button classes
const buttonClass = CSS_CLASSES.button.root; // "${CSS_CLASSES.button.root}"
const smallButton = CSS_CLASSES.button.sizes.sm; // "${CSS_CLASSES.button.sizes.sm}"`}
/>
