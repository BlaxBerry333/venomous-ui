# Typography Utilities

Pure CSS utility classes for typography styling. Use these classes directly on native HTML elements without any component wrappers.

## Quick Start

<StoryDocsTabCodeBlock
  tabs={[
    {
      label: "React",
      language: "tsx",
      code: `import "venomous-ui/core/styles/variables/index.css";
import "venomous-ui/core/styles/components/typography.css";

export function App() {
  return (
    <>
      <h1 className="venomous-title venomous-title--1">Page Title</h1>
      <p className="venomous-text venomous-text--secondary">
        Secondary paragraph text
      </p>
    </>
  );
}`,
    },
    {
      label: "Vue",
      language: "html",
      code: `<script setup lang="ts">
import "venomous-ui/core/styles/variables/index.css";
import "venomous-ui/core/styles/components/typography.css";
</script>

<template>
  <h1 class="venomous-title venomous-title--1">Page Title</h1>
  <p class="venomous-text venomous-text--secondary">
    Secondary paragraph text
  </p>
</template>`,
    },
  ]}
/>

## Text Truncation

### Single Line Ellipsis

Use `.venomous-ellipsis` to truncate text to a single line with ellipsis.

<StoryDocsCard>
  <div style={{ width: 300, border: "1px dashed #ccc", padding: 8 }}>
    <div className="venomous-text venomous-ellipsis">
      This is a very long text that will be truncated with an ellipsis when it exceeds the container width.
    </div>
  </div>
</StoryDocsCard>

<Source
  language="html"
  dark
  code={`<p class="venomous-text venomous-ellipsis" style="width: 300px">
  This is a very long text that will be truncated...
</p>`}
/>

### Multi-line Ellipsis (Line Clamp)

Use `.venomous-line-clamp` with preset classes or custom `--line-clamp` variable.

<StoryDocsCard>
  <StoryDocsFlex gap={24}>
    <div style={{ width: 200, border: "1px dashed #ccc", padding: 8 }}>
      <div style={{ fontSize: 12, marginBottom: 4, opacity: 0.6 }}>2 lines:</div>
      <div className="venomous-text venomous-line-clamp venomous-line-clamp--2">
        This is a long paragraph that demonstrates multi-line text truncation. The text will be cut off after 2 lines
        with an ellipsis at the end.
      </div>
    </div>
    <div style={{ width: 200, border: "1px dashed #ccc", padding: 8 }}>
      <div style={{ fontSize: 12, marginBottom: 4, opacity: 0.6 }}>3 lines:</div>
      <div className="venomous-text venomous-line-clamp venomous-line-clamp--3">
        This is a long paragraph that demonstrates multi-line text truncation. The text will be cut off after 3 lines
        with an ellipsis at the end. More content here to show the difference.
      </div>
    </div>
  </StoryDocsFlex>
</StoryDocsCard>

<Source
  language="html"
  dark
  code={`<!-- Using preset class -->
<p class="venomous-text venomous-line-clamp venomous-line-clamp--2">
  Long text truncated at 2 lines...
</p>

<p class="venomous-text venomous-line-clamp venomous-line-clamp--3">Long text truncated at 3 lines...</p>

<!-- Using custom line count -->

<p class="venomous-text venomous-line-clamp" style="--line-clamp: 4">
  Long text truncated at 4 lines...
</p>`}
/>

## Combining Classes

Classes can be freely combined for complex styling:

<StoryDocsCard>
  <div className="venomous-title venomous-title--2 venomous-text--theme venomous-text--center">
    Centered Theme Title
  </div>
  <div
    className="venomous-text venomous-text--sm venomous-text--secondary venomous-font--medium venomous-leading--relaxed"
    style={{ marginTop: 8 }}
  >
    This paragraph combines small size, secondary color, medium weight, and relaxed line height.
  </div>
</StoryDocsCard>

<Source
  language="html"
  dark
  code={`<h2 class="venomous-title venomous-title--2 venomous-text--theme venomous-text--center">
  Centered Theme Title
</h2>

  Combined styling example
</p>`}
/>

## TypeScript Usage

Access class names programmatically via `CSS_CLASSES`:

<Source
  language="tsx"
  dark
  code={`import { CSS_CLASSES } from "venomous-ui/core/constants";

// Title classes
CSS_CLASSES.typography.title // "venomous-title"
CSS_CLASSES.typography.titleSizes[1] // "venomous-title--1"

// Text classes
CSS_CLASSES.typography.text // "venomous-text"
CSS_CLASSES.typography.textSizes.sm // "venomous-text--sm"
CSS_CLASSES.typography.textColors.secondary // "venomous-text--secondary"

// Utility classes
CSS_CLASSES.typography.ellipsis // "venomous-ellipsis"
CSS_CLASSES.typography.lineClamp // "venomous-line-clamp"
CSS_CLASSES.typography.lineClampPresets[3] // "venomous-line-clamp--3"`}
/>
