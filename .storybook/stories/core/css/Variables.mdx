## CSS_VAR_NAMES Usage

Use `CSS_VAR_NAMES` with [`getCssVar()`](?path=/docs/core-helpers-helpers--docs#getcssvar) helper function to get the computed numeric value of a CSS variable at runtime:

<Source
  language="tsx"
  dark
  code={`import { CSS_VAR_NAMES } from "venomous-ui/core/constants";
import { getCssVar } from "venomous-ui/core/helpers";

// Get spacing value as number (e.g., 8)
const spacing = getCssVar(CSS_VAR_NAMES.spacingSm);

// With fallback
const duration = getCssVar(CSS_VAR_NAMES.durationNormal, 200);`}
/>
