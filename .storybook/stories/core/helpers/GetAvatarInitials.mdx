# getAvatarInitials

Extracts uppercase initials from a name string. Used by the Avatar component for text fallback display.

### Behavior by maxLength

<StoryDocsTable
  headers={["maxLength", "Input", "Output", "Rule"]}
  rows={[
    ["1", '"John Doe"', '"J"', "First character only"],
    ["2 (default)", '"John Doe"', '"JD"', "First + last word's first character"],
    ["2", '"Alice"', '"A"', "Single word returns one character"],
    ["2", '"John Michael Doe"', '"JD"', "First + last word (middle skipped)"],
    ["3", '"John Michael Doe"', '"JMD"', "First 3 words' initials"],
  ]}
/>
