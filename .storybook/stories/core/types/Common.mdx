# Common Types

Venomous UI exports common TypeScript types that are shared between React and Vue implementations. These types ensure consistency across frameworks.

All types use the `T` prefix for easy identification in code.

## TSize

<Source
  language="ts"
  dark
  code={`import type { TSize } from "venomous-ui/core/types";

type TSize = "sm" | "md" | "lg";`}
/>

Used for component sizing across buttons, inputs, modals, etc.

<StoryDocsTable
  headers={["Value", "Description"]}
  rows={[
    ["sm", "Small size"],
    ["md", "Medium size (default)"],
    ["lg", "Large size"],
  ]}
/>
