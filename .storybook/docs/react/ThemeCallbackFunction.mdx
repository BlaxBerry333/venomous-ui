# Theme Callback Function

When styling components with `style` prop, the attribute accepts both a standard CSS object and a theme callback functions.

Accessing the parameters of the theme callback function you can use the `theme` object.

<Source
  language="tsx"
  dark
  code={`<Button
  style={(theme) => ({
    backgroundColor: theme.mode.isDark
      ? theme.semanticColors.success.dark
      : theme.semanticColors.success.main,
    color: theme.fontColor("primary"),
    padding: \`\${theme.spacing(2)} \${theme.spacing(4)}\`,
    borderRadius: theme.borderRadius("medium"),
    boxShadow: theme.boxShadow("small"),
    transition: theme.transition("fast"),
  })}
  text="Themed Button"
/>`}
/>

## Theme object

<br/>

### Mode

Information about the current theme mode:

<Source
  language="tsx"
  dark
  code={`theme.mode.current   // "light" | "dark"

/>
