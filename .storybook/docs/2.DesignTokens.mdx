# Design Tokens

`venomous-ui` comes with a set of built-in design tokens that can be used to style your components.

You can access these design tokens from `venomous-ui/core/designs`.

<Source
  language="ts"
  dark
  code={`
import { DESIGN_TOKENS } from "venomous-ui/core/designs";
`}
/>

## Palettes

Pre-defined color palettes inspired by venomous creatures. Each has `light`, `main`, and `dark` variants.

<Source
  language="ts"
  dark
  code={`
const { light, main, dark } = DESIGN_TOKENS.palettesColorsResets.DART_FROG;
`}
/>

export const PalettePreview = () => {
  const palettes = DESIGN_TOKENS.palettesColorsResets;
  return (
    <Space.Grid columns={{ xs: 1, sm: 2, md: 3 }} spacing={4}>
      {Object.entries(palettes).map(([name, colors]) => (
        <Card key={name} variant="outlined" style={{ padding: "16px" }}>
          <div style={{ fontWeight: 600, marginBottom: "12px", fontSize: "14px", textAlign:"center" }}>
            {name}
          </div>
          <Space.Flex spacing={2} style={{ justifyContent: "center" }}>
            {["light", "main", "dark"].map((variant) => (
              <div key={variant} style={{ textAlign: "center" }}>
                <div
                  style={{
                    width: "48px",
                    height: "48px",
                    borderRadius: "8px",
                    backgroundColor: colors[variant],
                    marginBottom: "4px",
                  }}
                />
                <div style={{ fontSize: "10px", opacity: 0.7 }}>{variant}</div>
                <div style={{ fontSize: "10px", fontFamily: "monospace" }}>{colors[variant]}</div>
              </div>
            ))}
          </Space.Flex>
        </Card>
      ))}
    </Space.Grid>
  );
};

## Breakpoints

Breakpoints define responsive ranges for different device sizes. Each breakpoint has a `min` (range start) and `max` (range end) value.

- **min**: Use for `@media (min-width: ...)` media queries (mobile-first approach)
- **max**: Use for size constraints like Modal `maxWidth`

<Source
  language="ts"
  dark
  code={`
const { xs, sm, md, lg, xl } = DESIGN_TOKENS.breakpoints;

const { min, max } = xs;

const mediaQuery = \`@media (min-width: \${sm.min})\`;
`}
/>

export const BreakpointsPreview = () => {
  const breakpoints = DESIGN_TOKENS.breakpoints;
  return (
    <Space.Flex column spacing={3}>
      <Space.Flex spacing={4} style={{ alignItems: "center", fontWeight: 600, fontSize: "12px" }}>
        <div style={{ width: "80px" }}>Key</div>
        <div style={{ width: "80px" }}>Min</div>
        <div style={{ width: "80px" }}>Max</div>
      </Space.Flex>
      {Object.entries(breakpoints).map(([key, value]) => (
        <Space.Flex key={key} spacing={4} style={{ alignItems: "center" }}>
          <div style={{ width: "80px", fontFamily: "monospace", fontSize: "14px", fontWeight: 600 }}>{key}</div>
          <div style={{ width: "80px", fontFamily: "monospace", fontSize: "12px", opacity: 0.7 }}>{value.min}</div>
          <div style={{ width: "80px", fontFamily: "monospace", fontSize: "12px", opacity: 0.7 }}>{value.max}</div>
        </Space.Flex>

    </Space.Flex>
