# WorkflowBaseNode

`<WorkflowBaseNode>` is is a built-in component which is a basic component of workflow nodes.

You can think of it as a wrapper container with default styles, customize its content with the `children` attribute.

Use the built-in enum type `WorkflowNodeTypeDefault.Base` as its `NodeType`.
