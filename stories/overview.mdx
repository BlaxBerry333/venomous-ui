# Venomous UI

Venomous UI is a React component library that provides a set of reusable UI components.

<br />

## Installation

```shell
npm install git+https://github.com/BlaxBerry333/venomous-ui
```

## PeerDependencies

Make sure that the following dependencies are installed in advance.

```json
"peerDependencies": {
  "react": "^18.0.0 || ^19.0.0",
  "react-dom": "^18.0.0 || ^19.0.0",
  "zod": "^3.24.3"
},
```

## Import Modules

```jsx
import { Button } from 'venomous-ui';

function App() {
  return <Button text="Hello World" onClick={() => alert('hello world')} />;
}
```
