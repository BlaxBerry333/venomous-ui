## Examples

#### Default

It is not recommended to use it directly, the width defaults to `100%` of the parent element's width.

Prefer to place it in a parent element with a specified width, for example component `<Menu/>`.

<Canvas
  of={Stories.WithParent}
  sourceState="shown"
  source={{
    code: [
      "import { MenuItem } from 'venomous-ui';",
      '',
      'function App() {',
      '  return (',
      '    <>',
      "      {/* According to parent's width */}",
      '      <MenuItem label="Without Parent" sx={{ background: "pink" }} />',
      '',
      "      {/* According to <div/>'s 350px */}",
      '      <div style={{ width: 350 }}>',
      '        <MenuItem label="Inside Custom Element" sx={{ background: "orange" }} />',
      '      </div>',
      '',
      "      {/* According to <Menu/>'s 250px */}",
      '      <Menu',
      '        width={250}',
      '        items={[{ label: "Inside Menu Component" }]}',
      '        renderItem={(item) => <MenuItem label={item.label} sx={{ background: "skyblue" }} />}',
      '      />',
      '    </> ',
      '  );',
      '}',
    ].join('\n'),
  }}
/>

#### Ellipsis

By default, the `label` content expands to fit available space.

When the `ellipsis` property is `true`, text remains constrained within the parent element's width, text content that exceeds the width of the parent element will be truncated.
