## Examples

#### Default

It's recommended to use a `<span>` element as the container of the `<Tooltip>`'s child for these reasons:

1. **Solves event propagation issues**, such as preventing the tooltip from appearing when clicking on the child element
2. **Handles hover events for disabled elements**, such as `<button disabled>`
3. **Prevents styling and positioning problems**, such as the tooltip appearing behind other elements

```tsx
import { Tooltip, Card } from 'venomous-ui';

function App() {
  return (
    <Tooltip tooltip="Xxxxxxxxxxx">
      <span>
        <Card />
      </span>
    </Popover>

    // <Tooltip tooltip="Tooltip Text">
    //   <Card />
    // </Popover>
  );
}
```
